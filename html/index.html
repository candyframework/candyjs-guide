CandyJs

一个面向对象的优雅的 MVC and REST 框架

GitHub 文档

CandyJs 是一个 Node 框架，用于开发 Web 应用。CandyJs TypeScript 版本采用严格的 OOP 编写，并有着相对完善的新手指引文档

CandyJs 从诞生以来一直追求简洁可扩展的设计原则，在保持出色的性能和至简优雅代码的同时，也注重易用性。

DEMO
使用 CandyJs 你只需要从一个入口文件开始， 入口文件的内容可以使用 CandyJs 自带的工具来生成 
入口文件 index.js

var CandyJs = require('candyjs');

new CandyJs({
    'id': 1,
    
    // 定义调试应用
    'debug': true,
    
    // 定义应用路径
    'appPath': __dirname + '/app'
    
}).listen(8090, function(){
    console.log('listen on 8090');
});
OK 上面就是 Hello World 所需要的全部入口文件内容了 接下来看看文档吧
